Power Nodes

I designed these boards for that robot up there. These little "nodes" can be wired inline on whatever line and display a low res measurement of both voltage and current with the on-board seven-segment displays. Their main value comes in their ability to communicate over I2C. So, one master controller can request measurements from any node at any time. This provides a reliable way of monitoring our critiacla systems within the robot.

Using CNN's for Identifying Mining Material

This cool piece of python code takes audio input from a contact microphone that's attached near the digging mechanism of the Utah Student Robotics team's mining robot. It then classifies these sounds as either digging BP-1 (fine regolith simulant), BP-2 (course lunar ice simulant), or nothing at all. It does so by turning the sound into a spectrogram and performing convolutions on the image and feeding the result to a neural network along with information of the sound such as average pitch. (This code was done for a hackathon, so it's fairly messy)

Topological and Building STL Generator

Another fun hackathon project. On this one, a friend and I used Unity and some Google API's to get the topgraphical and building data for an area around a given set of latitude and longitude coords. After adjusting things such as exaggerated building size, and toggling buildings on and off, you could export the result to a stl format that could then be 3D printed.
